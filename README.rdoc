SETUP & RUN
---------------
* git clone git@github.com:psyxoz/blog.git
* cd blog && bundle install
* bundle exec rake db:setup
* bundle exec rails s
* open http://localhost:3000


USED GEMS
---------------
- Device (users authentication)
- DecentExposure (light controllers)
- Bootstrap-sass (twiter bootstrap with sass for styling)
- Resque (background jobs for emails sending)
- Friendly_id (seo tools)
- Redcarpet (markdown processing)
